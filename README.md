# Answer Exchange
A question and answer web app.

Created with [Nest](https://nestjs.com/) - a Node.js framework.

## Installation Steps:

```bash
$ npm install
```

* Add environment variables based on the .env.example file provided.
* Create a mysql database

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
